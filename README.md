
# Hotel Booking Analysis Dashboard (Power BI)

## 📊 Overview
This project analyzes hotel booking data to track revenue, cancellation trends, and customer behavior. The goal is to provide actionable insights for improving revenue and operational efficiency.

## 🚀 Tools & Technologies
- Python (Data Cleaning & Preprocessing)
- SQL (Data Extraction & KPI Calculation)
- Power BI (Dashboard Development & Visualization)
- DAX (KPI Measures)

## 📈 Key Metrics
- Total Revenue: 26M+
- Total Bookings: 119K+
- Cancellation Rate: 37%

## 🔍 Key Insights
- Online Travel Agencies (OTA) contribute the highest revenue share (~52%)
- Early bookings show the highest cancellation rates
- Peak revenue observed during mid-year months
- Higher average prices correlate with shorter stays

## 📊 Dashboard Features
- Revenue analysis by hotel type and market segment
- Monthly revenue trends
- Cancellation analysis by segment and booking type
- Price vs stay behavior insights
